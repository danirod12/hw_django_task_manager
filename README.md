# hw_django_task_manager

## Project start instruction
1. Clone this repo with
2. Join project directory
3. Start venv
4. Install requirements.txt
5. Migrate database
6. Start your server

### Windows example
1. `git clone https://github.com/danirod12/hw_django_task_manager/`
2. `cd .\hw_django_task_manager`
3. `.\venv\Scripts\activate`
4. `pip install -r requirements.txt`
5. `python manage.py migrate`
6. `python manage.py runserver`
